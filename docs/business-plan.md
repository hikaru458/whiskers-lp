# Whiskers 事業計画書（CAMPFIRE用）

## 1. プロジェクト概要

### 1.1 プロジェクト名
**Whiskers（ウィスカーズ）**

### 1.2 キャッチコピー
「企業とクリエイターをつなぐ コンテスト型UGCプラットフォーム」

### 1.3 プロジェクトの目的
小規模美容室・サロン・ブランド企業が、高品質なUGC（ユーザー生成コンテンツ）を低コストで獲得できるマッチングプラットフォームを構築する。

### 1.4 支援金の用途（目標金額：100万円）
| 用途 | 金額 | 詳細 |
|------|------|------|
| MVP開発費 | 40万円 | ダッシュボード・コンテスト機能実装 |
| インフラ費 | 20万円 | サーバー・ストレージ（1年分） |
| 営業活動費 | 25万円 | 資料作成・訪問営業交通費 |
| 運営準備金 | 15万円 | 規約作成・税務顧問・雑費 |

---

## 2. 背景と課題

### 2.1 市場の課題

**【企業側の課題】**
- 広告費が高騰（Instagram広告月10万円以上）
- 自社だけではUGCが作れない
- 依頼してもクリエイターとのマッチングが難しい
- クリエイターとの契約・管理が面倒

**【クリエイター側の課題】**
- 案件獲得が難しい（インフルエンサーでないと報酬が低い）
- スキル磨きの場がない
- ポートフォリオ作りの機会がない

### 2.2 市場規模
- 日本の美容・エステ市場：約3兆円
- コンテンツマーケティング市場：年間成長率15%
- UGC活用企業：大手企業から中小企業へ拡大中

---

## 3. 解決策：Whiskersの仕組み

### 3.1 サービスの流れ

```
【企業】                    【Whiskers】                    【クリエイター】
  │                              │                                  │
  │ ①コンテスト開催依頼         │                                  │
  │ ──────────────────────────> │                                  │
  │                              │                                  │
  │                              │ ②コンテスト告知・募集           │
  │                              │ ──────────────────────────────> │
  │                              │                                  │
  │                              │ ③作品投稿                      │
  │                              │ <────────────────────────────── │
  │                              │                                  │
  │ ④入賞作品のUGC利用権獲得    │                                  │
  │ <────────────────────────── │                                  │
```

### 3.2 ビジネスモデル

**【収益構造】**

| 項目 | 金額 | 内容 |
|------|------|------|
| 月額サブスク | 5,000円/月 | 企業のプラットフォーム利用料 |
| コンテスト賞金 | 20,000円〜 | 企業が設定（Whiskersは手数料なし） |
| 制作料 | 別途見積 | 優秀クリエイターへの直接発注 |

**【最初の3社を獲得した場合の収支例】**
- 月額サブスク：5,000円 × 3社 = 15,000円/月
- コンテスト賞金：20,000円 × 3社 = 60,000円（隔月開催想定で30,000円/月）
- **月間売上見込み：45,000円**
- **年間売上見込み：540,000円**

### 3.3 差別化ポイント

| 競合 | 課題 | Whiskersの解決策 |
|------|------|------------------|
| インスタグラム | 広告費高騰・リーチ低下 | コンテスト型で確実なUGC獲得 |
| クラウドソーシング | 品質バラバラ・管理コスト高 | 審査制で品質担保・一元管理 |
| インフルエンサー事務所 | 手数料高（30-50%）・大企業向け | 直接マッチング・中小企業価格 |

## 6. リスクと対策

### 6.1 想定されるリスク

| リスク | 内容 | 対策 |
|--------|------|------|
| 企業獲得の遅延 | 3社獲得に時間がかかる | 無料トライアル導入・営業強化 |
| クリエイター不足 | 参加者が集まらない | SNS告知・学生団体との提携 |
| クオリティ問題 | 作品レベルが低い | 審査基準の明確化・事前相談 |
| 副業制限 | 会社の規定に抵触 | 事前確認・副業届提出 |

### 6.2 法的リスク対策
- プライバシーポリシーの明確化
- 利用規約での責任範囲の限定
- 個人情報の適切な管理

## 9. まとめ

Whiskersは、**「予算が少ない美容・サロン業界に、確実なUGC獲得の仕組みを提供する」**ことを目指します。

**【なぜ支援が必要か】**
MVP開発に必要なクラウド費用・営業活動費をカバーし、
3社のパイロット企業を獲得するための初期投資が必要です。

**【支援の効果】**
- 企業側：低コストで高品質なUGCが獲得できる
- クリエイター側：実力を示せる場所と収入の機会
- 市場全体：活性化したコンテンツエコシステム

**【目標】**
3社のパイロット企業を獲得し、
コンテスト型UGCの新しいスタンダードを確立する。
